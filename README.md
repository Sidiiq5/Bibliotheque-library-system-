# Bibliotheque — Library Management System
 
A simple web-based library system built with HTML, CSS, and JavaScript.
 
## Files
 
| File | Description | Author |
|------|-------------|--------|
| `library-system.html` | Borrow & return books system | Sidiiq5 |
| `somali-library.html` | Somali books search page | bullhan66 |
 
## Features
 
- Browse and search books
- Borrow and return books
- Activity log
- Somali literature catalogue
## How to Use
 
Download any file and open it in your browser — no installation needed.
